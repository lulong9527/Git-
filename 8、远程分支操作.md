# 远程分支操作

## 1、相关命令

| 命令                                                | 描述                             |      |      |      |      |
| --------------------------------------------------- | -------------------------------- | ---- | ---- | ---- | ---- |
| git branch -a                                       | 查看远程分支个本地分支           |      |      |      |      |
| git push   origin branch_name                       | 推送本地分支到远程               |      |      |      |      |
| git push   origin :remote_branch(:远程分支名称)     | 删除远程分支（本地分支还保留）   |      |      |      |      |
| git checkout   -b local_branch origin/remote_branch | 拉取远程指定分支并在本地创建分区 |      |      |      |      |
